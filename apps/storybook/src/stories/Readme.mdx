# Next.js Demo

> Next.js demo web app

## Tech stack
- **Next.js**: React framework for building server-side rendered applications.
- **React**: JavaScript library for building user interfaces.
- **Next-intl**: Internationalization library for Next.js applications, providing support for translations and locale management.
- **Turborepo**: High-performance build system for JavaScript and TypeScript monorepos.
- **TypeScript**: Superset of JavaScript that adds static types.
- **shadcn/ui**: A set of accessible and customizable UI components built with Radix UI and Tailwind CSS.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **react-hook-form**: Library for managing form state and validation in React applications.
- **Zod**: TypeScript-first schema declaration and validation library.
- **vitest**: Fast and lightweight testing framework for JavaScript and TypeScript applications.
- **Storybook**: Tool for developing UI components in isolation.
- **eslint**: Linter for identifying and fixing problems in JavaScript and TypeScript code.
- **commitlint**: Linting tool for commit messages to enforce a consistent commit message format.
